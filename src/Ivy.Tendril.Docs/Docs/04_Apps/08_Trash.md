# Trash

<Ingress>
Deleted or duplicate-rejected plans land in **Trash** under `TENDRIL_HOME/Trash`—recoverable, not gone immediately.
</Ingress>

## Purpose

Removing a draft or failed run moves artifacts here so you can restore or read them later.

## UI

- **Filter** — By name, project, or text.
- **Open** — Read trashed markdown and history locally.

## Force Plan

If a plan was blocked as a duplicate but should run anyway, select it in Trash and use **Force Plan** to send the prompt through **MakePlan** again (bypasses duplicate checks).
