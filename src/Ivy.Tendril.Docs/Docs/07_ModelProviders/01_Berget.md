# Berget AI

<Ingress>
European AI infrastructure provider offering GLM and other models with full EU data residency.
</Ingress>

## Setup

1. Create an account at [console.berget.ai](https://console.berget.ai)
2. Run the setup wizard:
   ```bash
   npx berget code init
   ```
3. Launch OpenCode:
   ```bash
   opencode
   ```

The wizard handles API key configuration and provider setup automatically.

## Recommended Model

- **GLM 4.7** for coding tasks

Use `/models` inside OpenCode to browse and switch models.

## Using with Tendril

Select OpenCode as your coding agent during onboarding, or under **Settings > Configuration > Coding Agent**.
