# Scaleway

<Ingress>
European cloud provider with OpenAI-compatible Generative APIs supporting language, reasoning, and code models.
</Ingress>

## Setup

1. Create an account at [scaleway.com](https://www.scaleway.com) and generate a Secret Key via IAM
2. Launch OpenCode and connect:
   ```bash
   opencode
   ```
   Then type `/connect`, select **Scaleway**, and enter your API key.
3. Select a model with `/models`

## Using with Tendril

Select OpenCode as your coding agent during onboarding, or under **Settings > Configuration > Coding Agent**.
