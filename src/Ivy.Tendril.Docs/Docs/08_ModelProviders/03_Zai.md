# Z.AI

<Ingress>
Z.AI provides access to GLM models with a dedicated coding plan option.
</Ingress>

## Setup

1. Get an API key from the [Z.AI API Console](https://z.ai/manage-apikey/apikey-list)
2. Authenticate in OpenCode:
   ```bash
   opencode auth login
   ```
   Select **Z.AI** (or **Z.AI Coding Plan** if subscribed) and enter your API key.
3. Launch OpenCode and select a model with `/models`

## Recommended Model

- **GLM-4.7**

## Using with Tendril

Select OpenCode as your coding agent during onboarding, or under **Settings > Configuration > Coding Agent**.
